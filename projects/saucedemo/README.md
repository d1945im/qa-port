# Тестирование SauceDemo

**SauceDemo** (https://www.saucedemo.com/) — это учебный интернет-магазин, созданный для практики тестирования. Сайт содержит типовой функционал: авторизация, каталог товаров, фильтры, корзина, оформление заказа.

## Цели тестирования
- Проверить корректность работы страницы логина.
- Протестировать функциональность каталога (сортировка, добавление в корзину).
- Проверить процесс оформления заказа.
- Научиться оформлять баг-репорты на реальные дефекты.

## Инструменты
- **Chrome DevTools** — анализ сетевых запросов, проверка вёрстки.
- **GitHub** — хранение чек-листов, тест-кейсов, баг-репортов.
